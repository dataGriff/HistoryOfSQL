Inserting Data

Below is the template code for inserting data into table using SQL.

--This is a comment before inserting data
USE [DatabaseName];
GO

INSERT INTO [SchemaName].[TableName]
( [ColumnA],[ColumnB],[ColumnC] )
VALUES
( 'A', 1, '2015-04-01'),
( 'B', 2, '2015-04-02'),
( 'C', 3, '2015-04-03');

The template shows...

  • It is best practice to state the names of the columns you are going to insert into the table.
  • Dates and characters need to be inquotes but integers do not need to be.
  • Each value should be seperated by a comma.
  • Each row inserted should be contained within brackets and seperated by commas.
